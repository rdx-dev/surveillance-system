# Frontend - Micro Frontends

This folder contains all micro frontend applications using Webpack Module Federation.

## Structure

- **shell-app** (Port 3000) - Container/Host application
- **auth-mfe** (Port 3001) - Authentication micro frontend
- **live-feed-mfe** (Port 3002) - Live CCTV feed viewer
- **alert-mfe** (Port 3003) - Alert management
- **dashboard-mfe** (Port 3004) - Main dashboard
- **settings-mfe** (Port 3005) - Settings and configuration
